Java8实战-行为参数化

发表于 2019-12-20

[toc]

1. 行为参数化

一言以蔽之, 就是将行为传传递给一个方法, 从而实现代码整体逻辑的复用, 使代码更加的简洁,灵活

有点类似于匿名内部类, 但比他更加的简洁, 易读

2. 实现

  1. 需要一个函数式接口
  2. 用Lambda表示的行为

2.1 函数式接口

就是只定义一个抽象方法的接口(就算有许多的默认方法, 只要中定义了一个抽象方法, 就仍是函数式接口)

2.2 Lambda表达式

一种可以简洁的传递匿名函数的方式, 包含了 参数列表, 函数主体, 返回类型
主要语法为 : (参数) -> 主体 或 (参数) -> {主体}
